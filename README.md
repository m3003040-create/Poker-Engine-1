# Техасский Холдем покер с 5 ИИ

Полноценная игра в Техасский Холдем на чистом JavaScript, HTML и CSS. Работает в браузере, включая GitHub Pages. В игре участвуют 5 противников с разными стилями игры, полная логика торгов, банки, шоудаун и анимации.

## 🎮 Особенности

- **5 AI-противников** с различными стилями: тайтовый, агрессивный, случайный, адаптивный, LAG, TAG.
- **Игрок (человек)** — вы играете против компьютера.
- **Полная покерная механика**: префлоп, флоп, тёрн, ривер, шоудаун.
- **Управление банком**: основной банк, сайд-поты, распределение при all-in.
- **Настройки**: количество игроков (2-6), стартовые фишки, блайнды, агрессивность ИИ, скорость анимации, тема оформления.
- **Сохранение**: настройки сохраняются в localStorage.
- **Анимации**: сдача карт, перемещение фишек, подсветка победителя.
- **Логирование**: текстовый лог с фильтрацией и поиском.
- **Горячие клавиши**: F — фолд, C — чек, K — колл, R — рейз, A — all-in, Ctrl+S — сохранить, Ctrl+L — загрузить, H — справка.

## 🚀 Запуск

1. Склонируйте репозиторий или скачайте файлы.
2. Откройте `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. Настройте игру через кнопку ⚙️.
4. Нажимайте кнопки действий или используйте горячие клавиши.

## 🧠 Искусственный интеллект

Каждый AI имеет свой стиль, определяющий частоту блефа, агрессивность и диапазон рук:

- **Тайтовый** — играет только сильные руки, редко блефует.
- **Агрессивный** — часто рейзит, активно блефует, крадёт блайнды.
- **Случайный** — принимает случайные решения, непредсказуем.
- **Адаптивный** — учится на действиях оппонентов, подстраивается под их стиль.
- **LAG** — лузово-агрессивный, играет много рук агрессивно.
- **TAG** — тайтово-агрессивный, сильный стиль.

## ⚙️ Настройки

Через модальное окно (кнопка ⚙️) можно изменить:
- Количество игроков (2–6)
- Стартовые фишки
- Малый и большой блайнды
- Агрессивность ИИ (1–10)
- Скорость анимации
- Тему оформления
- Дополнительные опции (звук, анимации, подтверждения, горячие клавиши, компактный режим)

## 🎲 Правила игры

Классический Техасский Холдем:
1. Каждому игроку раздаётся по 2 закрытые карты.
2. Раунды торгов: префлоп → флоп → тёрн → ривер → шоудаун.
3. На флопе открываются 3 общие карты, на тёрне и ривере — по одной.
4. В шоудауне игроки составляют лучшую комбинацию из 5 карт (2 свои + 5 общих) и сравнивают.
5. Победитель забирает банк. При all-in создаются сайд-поты.
